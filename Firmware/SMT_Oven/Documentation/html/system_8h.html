void disableInterrupts ( )

Update SystemCoreClock variable.

Updates the SystemCoreClock & SystemBusClock variables with current core Clock retrieved from CPU registers. Disable interrupts

This function keeps a count of the number of times interrupts is enabled/disabled so may be called in recursive routines
