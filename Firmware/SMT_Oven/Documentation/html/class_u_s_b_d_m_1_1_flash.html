PartitionSel  partition = partition_flash0K_eeprom_all,
PartitionSplit  split = partition_default 
)
staticprotected

Initialise the EEPROM.

This function should be called before the first access to variables located in the eeprom.

Parameters
eepromEEPROM Data Size choice
partitionFlexNVM Partition choice (defaults to all EEPROM backing store)
splitSplit between A/B Flash portions (if supported by target)
Returns
FLASH_ERR_OK => EEPROM previous configured - no action required
FLASH_ERR_NEW_EEPROM => EEPROM has just been partitioned - contents are 0xFF, initialisation required
Note
This routine will only partition EEPROM when first executed after the device has been programmed.
