Parameters
cssck=> PCS assertion to SCK Delay Scaler
asc=> SCK to PCS negation delay
dt=> PCS negation to PCS assertion delay between transfers
ctarNum=> Index of CTAR register to modify

Note: Determines values for the smallest delay that is not less than specified delays.
