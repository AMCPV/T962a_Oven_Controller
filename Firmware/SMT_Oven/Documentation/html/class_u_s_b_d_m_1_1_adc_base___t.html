Detailed Description

template<class Info>
class USBDM::AdcBase_T< Info >

Template class representing an ADC.

Example

// Instantiate ADC channel adc0_se8
const AdcBase_T<adc0Info> Adc0;
// Initialise ADC
Adc0.setMode(resolution_16bit_se);
