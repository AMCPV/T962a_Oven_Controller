Detailed Description

template<PID::InFunction inputFn, PID::OutFunction outputFn, unsigned timerChannel>
class PID_T< inputFn, outputFn, timerChannel >

These template parameters connect the PID controller to the control variables/operations.

Template Parameters
inputFnInput function - used to obtain value of system state
outputFnOutput function - used to control the output variable
timerChannelThe PIT channel to use for timing
