static void USBDM::Mcg::hsRunMode ( bool  enable)
static

Switch to/from high speed run mode Changes the CPU clock frequency/1, and bus clock frequency /2 If the clock is set up for 120 MHz this will be the highest performance possible.

This routine assumes that the clock preferences have been set up for the usual RUN mode and only the Core clock divider needs to be changed.

Parameters
enableTrue to switch to HSRUN mode
