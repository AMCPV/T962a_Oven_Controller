virtual void USBDM::I2c::busHangReset ( )
pure virtual

Clear bus hang.

Generates I2C_SCL clock until I2C_SDA goes high.

This is useful if a slave is part-way through a transaction when the master goes away!

Implemented in USBDM::I2c_T< Info >.
